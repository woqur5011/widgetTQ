# TQQQ 전략 위젯 모음 — Android

TQQQ / QQQ 투자 전략을 홈 화면 위젯으로 한눈에 확인할 수 있는 안드로이드 앱입니다.

## 위젯 목록

| 위젯 | 설명 |
|------|------|
| **ALT TQ200** | TQQQ 200MA 기반 점수/비중 위젯 |
| **AGItQ** | TQQQ 200MA 골든크로스 전략 위젯 |
| **Snow** | Snowball TQQQ 전략 위젯 |
| **FGI** | Fear & Greed Index 위젯 |
| **QQQ 3/161/185** | QQQ 이동평균선 3/161/185 전략 위젯 |

## 빌드 방법

```
Android Studio에서 프로젝트를 열고 Run 실행
minSdk = 28 (Android 9+)
```

## 전략 출처 (투자 전략 레퍼런스)

아래 전략들을 코드로 구현한 개인 프로젝트입니다.

- TQQQ 200MA 전략 — 아기티큐
- TQQQ 200MA / QQQ 복합 전략 — 김쨰아빠
- Snowball 로직 — 호리오리
- QQQ 3/161/185 전략 — QQQ 3_161_185.js 레퍼런스

> 이 앱은 투자 조언이 아닙니다. 모든 투자 결정은 본인 책임입니다.
